# Namaz

A lightweight, cross-platform prayer times application for GNU/Linux and Windows.

## Overview

**Namaz** is a desktop application designed to provide accurate prayer times based on your precise location. Built with Electron, it combines a clean, modern interface with essential features like missed prayer tracking (Qaza), Azan notifications, and deep system integration for both Linux and Windows.

<p align="center">
  <img src="images/main.png" alt="Main Screen"/>
</p>

The application focuses on reliability and ease of use, ensuring that prayer timings are always accessible, whether as active notifications or through the system tray.

## Features

* **Precise Calculations**: Automatically determines accurate prayer times using location-based coordinates and timezones.
* **Qaza Tracker**: Built-in functionality to track and manage missed prayers.
* **Multimedia Alerts**: Plays the Azan (call to prayer) and provides desktop notifications.
* **Background Operation**: Runs efficiently in the system tray to ensure you never miss a notification.
* **Customization**: Fully configurable location settings (manual coordinates or city selection) and time formats.
* **Modern UI**: Native support for Dark Mode and a responsive, intuitive design.
* **Localization**: Multilingual support to cater to a global user base.

## Technical Details

The application leverages Electron's ability to provide a consistent experience across different operating systems. It uses local storage for privacy, ensuring your location data and prayer logs never leave your device.
